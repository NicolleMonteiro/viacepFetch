🚀 Sobre o projeto

O viacepFetch é uma aplicação que permite:

Inserir um CEP

Buscar automaticamente os dados na API do ViaCEP

Exibir informações como:

Logradouro

Bairro

Cidade

Estado

O objetivo do projeto é praticar:

✔️ Requisições com fetch

✔️ Manipulação de JSON

✔️ Manipulação do DOM

✔️ Tratamento de erros

✔️ Programação assíncrona (async/await)

🛠️ Tecnologias utilizadas

HTML5

CSS3

JavaScript

Fetch API

API pública do ViaCEP

🔗 API utilizada

ViaCEP: https://viacep.com.br/

💡 Funcionalidades

Validação básica de CEP

Mensagem de erro para CEP inválido

Preenchimento automático dos campos

Interface simples e intuitiva
